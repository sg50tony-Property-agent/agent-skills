# Property Video Script Creator

## Description
Create short property video scripts for real estate listings.

Scripts are designed for TikTok, Instagram Reels and YouTube Shorts.

Video duration: 30–60 seconds.

Structure: Hook → Walkthrough → Highlights → Call to Action.

## Trigger Phrases

create property video script
generate condo video script
write real estate reel script
create property walkthrough script

## Example Prompt

Create a 45 second video script for:

Property: 4 Room HDB
Location: Tampines
Size: 92 sqm
Highlights: renovated, near MRT, high floor

## Output

Hook

Attention grabbing opening line.

Walkthrough

Describe living room, kitchen, bedrooms.

Key Highlights

Connectivity, amenities, lifestyle benefits.

Closing Call To Action

Contact Tony Ong for viewing today.
