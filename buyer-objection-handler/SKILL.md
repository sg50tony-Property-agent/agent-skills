# Buyer Objection Handler

## Description
Generate persuasive responses to common buyer objections in Singapore property transactions.

Responses should be professional, reassuring and informative.

## Trigger Phrases

handle buyer objection
respond to buyer concern
property sales objection response
real estate buyer objection

## Example Prompts

Buyer says price is too high.

Buyer says the market might drop.

Buyer says the location is not good.

Buyer is worried about interest rates.

## Output

Objection

State the buyer’s concern.

Response Strategy

Explain how the agent should address the concern.

Suggested Response

Provide a clear and persuasive response suitable for conversation with the buyer.
