# Property Listing Generator

## Description
Generate persuasive property listing descriptions for Singapore real estate listings.

Focus on highlighting:
- Unique selling points
- Lifestyle benefits
- Nearby amenities
- Investment potential

Output should include:
Headline, Key Highlights, Description, Nearby Amenities and Call To Action.

## Trigger Phrases

create property listing  
generate property description  
write condo listing  
write HDB listing  
create real estate listing  

## Example Prompt

Generate a property listing for:

Property: 3 Bedroom Condo  
Location: Tanjong Pagar  
Size: 980 sqft  
Highlights: MRT nearby, high floor, city view
