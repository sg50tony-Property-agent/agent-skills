# Social Media Property Post Generator

## Description
Generate engaging social media captions for Singapore property listings.

Create optimized posts for Facebook, Instagram and TikTok.

Tone should be exciting, persuasive and marketing-focused.

## Trigger Phrases

create property social post
generate property marketing caption
write condo social media post
create real estate instagram caption

## Example Prompt

Create a social media post for:

Project: The Sen
Unit: 3 Bedroom
Size: 1001 sqft
Price: 2.65M
Highlights: near MRT, city fringe, new launch

## Output

Facebook Post

Longer storytelling caption with highlights and call to action.

Instagram Caption

Shorter caption with emojis and hashtags.

TikTok Caption

Short catchy caption.

Call To Action

Contact Tony Ong for viewing today.
