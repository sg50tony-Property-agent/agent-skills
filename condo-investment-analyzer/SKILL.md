# Condo Investment Analyzer

## Description
Analyze Singapore condominium investment potential using price, size, psf and estimated rental income.

The skill calculates rental yield and evaluates investment attractiveness.

## Trigger Phrases

analyze condo investment  
calculate rental yield  
evaluate property investment  
condo investment analysis  

## Example Input

Property Price: 1650000  
Size: 710 sqft  
PSF: 2323  
Estimated Monthly Rent: 5200  
Project: Marina Bay Residences  

## Output

Property Summary

Project Name  
Price  
Size  
PSF  

Rental Analysis

Estimated Monthly Rent  
Annual Rental Income  
Gross Rental Yield  

Investment Potential

Market Position  
Tenant Demand  
Future Growth Potential  

Target Buyer Profile

Investor  
Owner-investor  
Long term buyer
